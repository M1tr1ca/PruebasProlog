
    ```sh
    cd PruebasProlog
    ```
3. Abre los archivos `.pl` con SWI-Prolog y ejecuta los ejemplos.

## Uso

Iniciar:
```bash
swipl
```


Para ejecutar un archivo Prolog, abre SWI-Prolog y carga el archivo:
```prolog
?- [nombre_del_archivo].
```

Luego, puedes ejecutar las consultas definidas en el archivo.
